Go

University-branded URL shortening

What is Go?

Go provides University-branded URL shortening for student organizations, administration, and ULife groups. Any approved user can generate a go.gmu.edu address that can redirect to a longer URL.

Go is a product of the George Mason University Student-Run Computing and Technology student organization.

Why should I use Go?

Go features:

  • Data tracking on the number of clicks a Go link gets.
  • Options to share Go links across various social media platforms.
  • QR Code generation at various resolutions, ready to be shared.
  • Optional expiration for links after particular times.

How do I access Go?

In order to prevent abuse of the URL shortner, access to Go is moderated by SRCT administrators. New users will need to fill out a registration form in order to become an approved user. Additionally, only people with a current, valid Mason username and password may request to be approved.
